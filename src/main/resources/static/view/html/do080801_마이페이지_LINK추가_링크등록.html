    추가등록

    등록한 관련 과제/논문 정보는 관리자의 승인을 통해 서비스됩니다. 한건만 등록하신다면 “승인요청”버튼을, 한번에 더 많은 과제/논문을 등록하고 싶다면 “추가등록”버튼을 클릭해주세요.

    추가등록시 입력하신 정보는 하단부 리스트에 추가됩니다. 여러정보를 한번에 신청하실경우 리스트 아래 “일괄 승인요청”버튼을 클릭해주세요.
